# Arch OS Rice using:

- Terminal: Alacritty
- Music player: ncmpcpp
- File manager: Nautilus (+ nautilus-terminal)
- Window manager: bspwm
- Compositor: picom/compton
- Shortcuts: sxhkd
- Lockscreen: Better LockScreen
- Background: nitrogen (+ mountains.png)
- App launcher: Rofi
- Status bar: Polybar (elementary theme)
